# FestRecipe 🎸
> 페스티벌 타임테이블 및 지능형 예상 셋리스트 재생목록 생성 서비스

FestRecipe는 인디음악 및 대중음악 페스티벌의 타임테이블을 시각화하고, 아티스트들의 과거 공연 데이터를 바탕으로 예상 셋리스트를 제공하며, 이를 유튜브 재생목록(Playlist)으로 즉시 감상할 수 있도록 도와주는 웹 애플리케이션입니다.

---

## 주요 기능 🌟

1. **페스티벌 라인업 공개 단계별 대응**
   - **1단계 (전체 라인업):** 요일 구분이 없는 전체 아티스트 리스트업
   - **2단계 (일별 라인업):** 일자별 아티스트 분할 표기
   - **3단계 (타임테이블):** 시간/스테이지별 세로형 타임라인 뷰 제공 (화면 너비에 맞춰 유연하게 반응하며 잘림 방지 처리)

2. **지능형 예상 셋리스트 & 라이브 아카이브**
   - 최근 공연 데이터 기반의 예상 셋리스트 구성 (발매곡, 미발매곡, 커버곡 등 곡 유형 표기)
   - 각 곡의 공식 음원(YouTube) 및 지난 페스티벌/단독콘서트의 **풀캠 타임스탬프 딥링크** 및 팬 녹화 라이브 클립 제공

3. **심리스한 멀티트랙 재생 (YouTube watch_videos)**
   - **예상 셋리스트 전체 재생:** 한 번의 클릭으로 셋리스트의 모든 공식 음원/MV를 연속으로 감상할 수 있는 유튜브 재생목록 링크 동적 생성
   - **특정 곡부터 재생 시작:** 선택한 곡에서부터 시작하여 나머지 곡들이 연속 재생되도록 순서가 맞춤 재배열된 딥링크 제공

## 기술 스택 🛠️

- **Core:** React (TypeScript) + Vite
- **Styling:** Vanilla CSS (CSS Variables 기반 테마 및 모던 인터랙션 설계)
- **Routing:** React Router DOM
- **Build & Lint:** ESLint, TypeScript Compiler (`tsc`)

---

## 디렉토리 구조 📂

```
src/
├── components/          # 공통 UI 컴포넌트 (타임테이블 그리드 등)
├── pages/               # 주요 페이지 (페스티벌 목록, 상세 정보)
├── services/            # 페스티벌 및 아티스트 데이터 공급용 모크 서비스
├── types/               # 공통 TypeScript 타입 정의
├── index.css            # 글로벌 디자인 시스템 및 CSS 변수 테마
└── main.tsx
```

## 실행 및 빌드 방법 🚀

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```
빌드가 완료되면 `/dist` 폴더에 프로덕션용 정적 파일들이 생성됩니다.
